## RISC-V Math (HarmonyOS Symbolic Calculator)这是一个基于 OpenHarmony (HarmonyOS Next) 开发的符号数学计算器应用。项目对标安卓平台的 **Maple Calculator**，旨在为用户提供从公式输入到深度数学分析的完整体验。

应用结合了本地 ArkTS 高性能 UI 与云端大模型（DeepSeek）的推理能力，实现了复杂的数学公式识别与分步求解。

### 核心功能 
### 1. 专业的 LaTeX 公式编辑 **自定义数学键盘**：针对移动端优化的键盘布局，支持常用数学符号（积分、求和、三角函数等）的快捷输入。
* **实时渲染**：通过 WebView 集成 KaTeX 引擎，实现输入过程中的公式实时、高保真渲染，支持光标定位与编辑。

### 2. 实时结果预览 内置轻量级计算引擎，在用户输入的同时尝试进行本地计算。
* 支持基础的数值计算与简单的符号化简结果预览。

### 3. AI 深度数学分析 **云端推理**：集成了 **DeepSeek** 大模型 API。
* **结构化输出**：不局限于单纯的文本对话，而是通过 Prompt Engineering 让 AI 输出严格的 JSON 数据。
* **分步求解**：提供完整的解题步骤（Step-by-step）、定义域/值域分析、零点判定等属性。
* **图文混排渲染**：分析结果页采用 WebView 渲染，完美支持数学公式与中文讲解的混合排版。

## 技术架构 **前端**：ArkTS, ArkUI, WebView (KaTeX)
* **后端/云服务**：华为 AppGallery Connect (AGC) 云函数
* **AI 模型**：DeepSeek Chat (V3)
* **通信**：通过 AGC 云函数代理请求，解决跨域与签名问题，确保在云端模拟器环境下的连通性。

## 待完善功能 (Future Work)目前项目处于核心功能验证阶段，以下模块计划在后续版本中支持：

1. **矩阵计算**：目前暂不支持矩阵的输入与行列式/逆矩阵运算。
2. **微分方程优化**：微分方程的输入体验尚需优化（如 y', y'' 的快捷输入）。
3. **字母键盘**：目前无法直接输入希腊字母如Ω/ω，τ，Φ/φ。
4. **函数图像绘制**：尚未集成 Canvas 或 ECharts 进行函数曲线的绘制与交互。

## 运行说明1. 使用 **DevEco Studio** 打开项目。
2. Sync Project 确保依赖下载完成。
3. 项目内置了体验用的 API Key（通过 AGC 代理），可直接在模拟器或真机上运行。
4. 如需使用自定义 API Key，可在应用右上角“设置”中开启“自定义模式”并填入 DeepSeek Key。
